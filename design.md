## 一、产品概述

### 1.1 产品名称
**海龟交易策略实验室（Turtle Strategy Lab）**

### 1.2 产品定位
一款面向量化实习学员的**浏览器端交互式策略可视化与回测工具**。用户无需安装任何环境，打开 `index.html` 即可：切换标的、调节策略参数、观察 Donchian 通道与 ATR、查看买卖信号、运行回测并获得 MDD / Sharpe / Cumulative Return 等核心指标。

### 1.3 核心价值
- **看得懂**：把海龟策略的通道突破、ATR 止损等抽象概念变成可交互的图表。
- **调得动**：参数（通道周期、ATR 周期、止损倍数）实时可调，即时重算重绘。
- **评得清**：一键回测，量化指标面板 + 回测对比图 + 交易记录表三视角评估。

### 1.4 设计原则
1. **零依赖运行**：纯 HTML + CSS + JS，双击打开即用，无需后端/构建工具。
2. **涨红跌绿**：遵循中国股市配色惯例。
3. **过程可追溯**：所有计算（通道/ATR/信号/回测）前端手写，中间结果可展示。
4. **轻量专注**：单页应用，一个屏幕内完成"选股 → 调参 → 看图 → 回测"闭环。

## 四、信息架构

```
海龟交易策略实验室（单页）
├── 顶部栏 Header
│   ├── 产品标识 + 标题
│   └── 标的快速切换 Tabs（中芯国际 / 比亚迪 / 长江电力 / 兆威机电）
├── 左侧控制面板 Sidebar
│   ├── 标的选择列表
│   └── 策略参数区（滑块 + 输入框 + 应用按钮）

    ├── ① 策略主图（价格 + 通道 + ATR + 买卖标记）
```

## 六、交互设计

### 6.1 核心用户流程

```
打开 index.html
   │
   ▼
默认加载第一支股票（中芯国际）→ 自动计算通道/ATR/信号 → 渲染主图+指标
   │
   ├── 点击标的 Tab / 列表项 ──► 切换股票，全部重算重绘
   │
   ├── 拖动参数滑块 / 输入数值 ──► 点击"应用" ──► 重新计算信号+回测 ──► 刷新图表与指标
   │
   ├── 主图 dataZoom 拖选 ──► 联动放大某段时间
   │
   └── 悬停图表 ──► tooltip 显示日期/价格/通道/ATR/信号
```

### 6.2 交互细节
- **参数变更**：滑块拖动时实时显示数值；点击"应用参数"才触发重算（避免频繁重算卡顿）。
- **标的切换**：瞬时切换，保留当前参数设置。
- **图表 tooltip**：主图 tooltip 同时展示 close/upper/lower/atr/signal。
- **指标卡片**：累计回报为正→红、为负→绿；MDD 始终绿（回撤为负）。
- **空数据兜底**：若某标的数据不足（< 60 条），主图区显示提示"数据不足，请增加获取天数"。

## 八、数据流设计

```
data/*.json ──► load_stock_data(code) ──► 日线数组
                                              │
                ┌─────────────────────────────┤
                ▼                             ▼
   calculate_donchian(data,N,M)     calculate_atr(data,n)
        │  upper/lower                   │  tr/atr
        └──────────┬──────────────────────┘
                   ▼
        generate_signals(data, stop_k)
            │  signal / stop_price
            ▼
        run_backtest(data, capital, commission)
            │  daily_equity / trades / metrics
            ▼
   ┌───────────────┬───────────────┬─────────────┐
   ▼               ▼               ▼             ▼
 主图渲染      指标卡片渲染     回测对比图      交易记录表
```

所有计算在前端 JS 完成，无网络请求（除 ECharts CDN）。

## 九、技术方案

| 项 | 选型 |
|----|------|
| 结构 | 单文件 `index.html` + `css/` + `js/` |
| 图表 | ECharts 5.5（CDN: cdn.jsdelivr.net） |
| 策略引擎 | 纯 JS 手写（Donchian / ATR / 信号 / 回测） |
| 数据 | 本地 JSON（`data/*.json`） |
| 样式 | 原生 CSS（无框架） |
| 构建 | 无需构建，浏览器直开 |

> 禁止使用第三方回测框架（backtrader 等），保证计算过程透明可追溯。
